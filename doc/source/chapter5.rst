5.2 еәҰйҮҸж•°жҚ®зҡ„жё…жҙ—е’ҢеӨ„зҗҶ
========================

5.2.1 ж•°жҚ®жё…жҙ—
--------------

еҺҹе§ӢеәҰйҮҸж•°жҚ®еҸҜиғҪеҢ…еҗ«еҷӘеЈ°е’ҢејӮеёёпјҢйңҖиҰҒжё…жҙ—:

- **еҺ»йҮҚ**: еҺ»йҷӨйҮҚеӨҚзҡ„еәҰйҮҸж•°жҚ®зӮ№
- **иЎҘеҖј**: еЎ«е……зјәеӨұзҡ„ж•°жҚ®зӮ№ (зәҝжҖ§жҸ’еҖјгҖҒеүҚеҖјеЎ«е……)
- **еҺ»еҷӘ**: дҪҝз”Ёз§»еҠЁе№іеқҮжҲ–дёӯдҪҚж•°ж»ӨжіўеҺ»йҷӨеҷӘеЈ°
- **ејӮеёёжЈҖжөӢ**: иҜҶеҲ«е№¶ж Үи®°ејӮеёёж•°жҚ®зӮ№

5.2.2 ж•°жҚ®еӨ„зҗҶ
--------------

еёёз”Ёзҡ„ж•°жҚ®еӨ„зҗҶж–№жі•:

- **з§»еҠЁе№іеқҮ (Moving Average)**: е№іж»‘ж•°жҚ®жіўеҠЁ
- **жҢҮж•°еҠ жқғз§»еҠЁе№іеқҮ (EWMA)**: иҝ‘жңҹж•°жҚ®жқғйҮҚжӣҙй«ҳ
- **е·®еҲҶ (Derivative)**: и®Ўз®—еҸҳеҢ–зҺҮ
- **з§ҜеҲҶ (Integral)**: и®Ўз®—зҙҜз§ҜйҮҸ
- **зҷҫеҲҶдҪҚи®Ўз®—**: и®Ўз®— P50, P90, P95, P99

**Python (Pandas)**

.. code-block:: python

   import pandas as pd

   # з§»еҠЁе№іеқҮ
   df['latency_ma'] = df['latency'].rolling(window=5).mean()

   # жҢҮж•°еҠ жқғз§»еҠЁе№іеқҮ
   df['latency_ewma'] = df['latency'].ewm(span=5).mean()

   # зҷҫеҲҶдҪҚи®Ўз®—
   p50 = df['latency'].quantile(0.50)
   p90 = df['latency'].quantile(0.90)
   p99 = df['latency'].quantile(0.99)

**Go (е®һж—¶жөҒеӨ„зҗҶ)**

.. code-block:: go

   // Go дёӯдҪҝз”Ёж»‘еҠЁзӘ—еҸЈи®Ўз®— P99
   type LatencyTracker struct {
       mu      sync.Mutex
       window  []float64
       maxSize int
   }

   func (t *LatencyTracker) Record(latency float64) {
       t.mu.Lock()
       defer t.mu.Unlock()
       t.window = append(t.window, latency)
       if len(t.window) > t.maxSize {
           t.window = t.window[1:]
       }
   }

   func (t *LatencyTracker) Percentile(p float64) float64 {
       t.mu.Lock()
       sorted := make([]float64, len(t.window))
       copy(sorted, t.window)
       t.mu.Unlock()
       sort.Float64s(sorted)
       idx := int(float64(len(sorted)-1) * p)
       return sorted[idx]
   }

5.6 жң¬з« е°Ҹз»“
============

жң¬з« д»Ӣз»ҚдәҶеәҰйҮҸж•°жҚ®д»ҺйҮҮйӣҶеҲ°еұ•зӨәзҡ„е®Ңж•ҙжөҒзЁӢ:

- еәҰйҮҸж•°жҚ®зҡ„иҒҡеҗҲж–№ејҸе’Ңж—¶еәҸж•°жҚ®еә“йҖүеһӢ
- ж•°жҚ®жё…жҙ—е’ҢеӨ„зҗҶзҡ„еёёз”Ёж–№жі•
- еәҰйҮҸеҸҜи§ҶеҢ–зҡ„еӣҫиЎЁзұ»еһӢе’Ңд»ӘиЎЁзӣҳи®ҫи®ЎеҺҹеҲҷ
- еӣӣз§Қдё»жөҒзҡ„еәҰйҮҸжҠҖжңҜж Ҳ: TIGгҖҒELKKгҖҒPrometheusгҖҒOpenTelemetry

.. note::

   **е…ій”®иҰҒзӮ№:**

   - ж—¶еәҸж•°жҚ®еә“жҳҜеәҰйҮҸж•°жҚ®еӯҳеӮЁзҡ„йҰ–йҖү
   - дҪҝз”ЁзҷҫеҲҶдҪҚж•°иҖҢйқһеқҮеҖјжқҘеұ•зӨәе»¶иҝҹ
   - д»ӘиЎЁзӣҳеә”иҜҘеұӮж¬Ўжё…жҷ°гҖҒдёҖзӣ®дәҶз„¶
   - OpenTelemetry жӯЈеңЁжҲҗдёәеҸҜи§ӮжөӢжҖ§зҡ„з»ҹдёҖж ҮеҮҶ
